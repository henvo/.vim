### Prerequisites
* git
* ripgrep
* FZF

``` bash
# On Ubuntu/Debian
apt install git ripgrep fzf

# On Mac (brew)
brew install ripgrep fzf
```

### Installation

``` bash
git clone --recurse-submodules git@github.com:henvo/.vim.git
```

### Updating plugins
``` bash
git submodule update --recursive --remote
```
